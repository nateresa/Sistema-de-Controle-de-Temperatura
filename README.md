# 🌡️ Sistema de Controle de Temperatura

Este é um projeto simples em Java que simula um sistema de controle de temperatura ambiente via terminal.

O usuário pode visualizar, aumentar, diminuir e definir a temperatura, com validações de limites e mensagens de alerta.

---

## 📌 Funcionalidades

- Visualizar a temperatura atual
- Aumentar a temperatura
- Diminuir a temperatura
- Definir uma temperatura manualmente
- Validação de limites:
  - Temperatura mínima: **16°C**
  - Temperatura máxima: **31°C**
- Alertas automáticos:
  - Ambiente muito frio
  - Ambiente muito quente
  - Temperatura fora do permitido

---

## 🧠 Lógica aplicada

Este projeto utiliza conceitos básicos de programação:

- Estrutura de repetição (`while`)
- Estruturas condicionais (`if/else`)
- Entrada de dados com `Scanner`
- Controle de fluxo com menu interativo
- Validação de limites (mínimo e máximo)

---

## 🖥️ Como executar

1. Abra o projeto em uma IDE (ex: IntelliJ IDEA)
2. Compile e execute a classe `temperatura`
3. Interaja com o menu exibido no terminal

---

## 📋 Menu do sistema

O sistema apresenta as seguintes opções:

1. Ver temperatura atual  
2. Aumentar temperatura  
3. Diminuir temperatura  
4. Definir temperatura  
5. Sair  

---

## ⚠️ Regras do sistema

- A temperatura não pode ser menor que **16°C**
- A temperatura não pode ultrapassar **31°C**
- O sistema ajusta automaticamente valores fora do limite
- Mensagens são exibidas conforme o estado do ambiente
